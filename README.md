# Homebrew Tap for AgentGazer

This is the official Homebrew tap for [AgentGazer](https://github.com/agentgazer/agentgazer).

## Installation

```bash
brew install agentgazer/tap/agentgazer
```

Or:

```bash
brew tap agentgazer/tap
brew install agentgazer
```

## Usage

```bash
agentgazer              # Launch server + proxy + dashboard
agentgazer onboard      # First-time setup
agentgazer --help       # Show all commands
```
